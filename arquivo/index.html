Um arquivo é um recurso utilizado para armazenamento de informação. Normalmente ligado a um programa, o arquivo contém tipo, tamanho e outros atributos, como data de criação por exemplo.
Um arquivo pode ser um slide, um pdf, um txt, uma especificação de trabalho, um vídeo, uma música, etc.
Um arquivo pode ser usado para disponibilizar recursos para os alunos. Recursos estes englobam, materiais didáticos, especificações de trabalho, livros, apostilas, artigos, etc.
Para adicionar um arquivo, você só precisa seguir este tutorial: Tutorial Arquivo
